# Panorama-Swap-INK-SC

![Panorama logo](https://i.imagesup.co/images2/302ff85b1ff055738b8c63ae2ca0137f9c3a6929.png)







The following repository is used to store all Panorams swaps's smart contracts.

Panorama Swap is a decentralized exchange built on Aleph Zero.
Our vision is to deliver a highly capable system for the permissionless trading of digital assets. the state of the art consensus algorithm pioneered by Aleph Zero ensures lightning fast settlement, as well as simple and efficient liquidity provision.

To learn more about Panorama Swap, please visit our website:
https://panoramaswap.com/

We also have more in-depth documentation and explanation about the DEX logic in the following link:
https://panoramaswap.gitbook.io/product-docs/

We are using INK! and RUST to write our smart contracts, more info on INK! can be found right here:
https://paritytech.github.io/ink/
